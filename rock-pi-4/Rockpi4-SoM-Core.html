    ROCK Pi 4 >  ROCK Pi 4 Core SoM

Introduction

ROCK Pi 4 Core is an SoM(System on Module) by Radxa based on Rockchip RK3399 SoC in a stamp form factor at 55mm x 55mm size, integrating CPU/DRAM/STORAGE. ROCK Pi 4 Core offers out of box cost-effective solution for multimedia and AI applications, accelerates customer's product development. It has the following features:

Hexa core 64 bits High Performance Solution

ROCK PI 4 Core is powered by Rockchip RK3399 SoC with 64bits Dual Cortex A72 high performance core up to 1.8GHz and Quad Cortex A53 low power core up to 1.4Ghz, up to 4GB RAM and up to 128GB eMMC. With a simple carrier board(2 layer base board is possible with all functions), the customer can quickly put the product into production.


Rich interfaces

Support I2C, SPI, UART, ADC, PWM, GPIO, Ethernet, PDM, USB2, USB3, I2S, MIPI, eDP as well as PCIe 2.0 high bandwidth buses.

Display capability

Dual Video Output Processor, support dual display output to any two of HDMI, eDP, MIPI, DP, resolution up to one 4K x 2K with one 2.5K.


Powerful multimedia utility

Support 4K VP9 and 4K 10bits H264/H265 video decoding, up to 60FPS

Support multiple 1080P video formats decoding including VC-1, MPEG-1/2/4, VP8

Support 1080P encoding to H.264, VP8 formats

Small size and low power

Custom 200PIN stamps form factor at 55mm x 55mm size, saving board space.

Support multiple OS

Support Ubuntu 18.04 / Debian 10 / Buildroot / Android.

Open documents and source code

Source code, documents, tool&utilities are freely available, community and commercial support to help you put prototype into production

Widely used in varies applications

Ideal SoM for Robotics, HMI, Retail Machines, Smart Home, IOT Gate Way, Industrial Controls, Medical Equipments etc.

Specification

Core System
SoC Rockchip RK3399
CPU Dual Cortex-A72@1.8GHz, Quad Cortex-A35@1.4GHz
GPU Mali T860MP4
RAM Optional 1GB, 2GB, 4GB LPDDR4
Storage Optional 16GB, 32GB, 64GB, 128GB high performance eMMC
Interfaces
Display - 1x HDMI up to 4K x 2K@60HZ
- 1x eDP four lanes, 2.7Gps per lane
- 2x MIPI four lanes, 1.6Gbps per lane
- 1x DP, support up to 4K x 2K@60HZ
Touch Support USB/I2C capacitive touch panel
Audio - LINEOUT
- I2S
- PDM, support mic array
SD MMC 2x SDIO support SD card or SDIO devices
Ethernet 1000M ethernet with phy
USB - Up to 2x full function USB C
- Up to 2x HOST 3.0
- Up to 2x USB HOST 2.0
- 1x USB 2.0 OTG or 1x USB 3.0 OTG
Camera Up to 2x MIPI CSI, dual ISP, 1x max 13Mpx sensor or 2x 8M sensor
PCIe 1x PCIe x4, support NVMe SSD or PCIe cards
UART Up to 3x UARTs
PWM Up to 4x PWM
IIC Up to 5x IIC
ADC Up to 2x ADC
SPI Up to 2x SPI
Battery Support 3.7V lion battery
Electronic

3.3V Output 3.3V/1A
Operation Temperature 0~80 degree, optional wider temperature range RK3399K
Storage Temperature -20~60 degree
Dimension
